NAME

srm_content - manipulate SRM/V10 webstore content

VERSION

version 0.120

USAGE

    % srm_content help

DESCRIPTION

This is the command line front-end to GSI::SRM::Content::Cmd, a suite of tools for manipulating static content in GSI's SRM/V10 webstores.

CONFIGURATION

Please consult the documentation for each subcommand under GSI::SRM::Content::Cmd::Command for configuration options.
